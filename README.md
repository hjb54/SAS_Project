### **The Titanic: A Statistical Analysis of Age, Sex, and Class of Passengers**
This project examines survival outcomes from the Titanic dataset using statistical methods in SAS. The analysis focuses on how age, sex, and passenger class influenced survival rates, supported by exploratory data analysis, visualizations, and logistic regression modeling.

* Paper: STAT 303 Final Paper.pdf - full write-up of the analysis, results, and interpretation.
* Code: EDA.sas - includes data cleaning, decriptive statistics, visuaizations, and logistic regression modeling.

The project was completed as part of a final project for STAT 303 and demonstrated proficiency in SAS programming, statistical reasoning,  exploratory data analysis, and clear communication of analytical findings..
